## Description

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# docker
$ docker-compose up
```

## Test

```bash
# unit tests
$ npm run test:unit

# e2e tests
$ npm run test:integration

# all tests
$ npm run test

# test coverage
$ npm run test:cov
```
